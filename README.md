# Note taker with Docopt

Take your notes like a pro
---

### To Start

```
Run pipenv shell

```

##### To see all usage with docopt, run:

```
python3 app.py
```

#### For help with the commands. run:

```
python3 app.py help
```

#### To View all the notes:

```
python3 app.py view
```
---

#### To Create a new Note:

```
python3 app.py create "title" "the note"

```
